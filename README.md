# matrix-appservice-discord
A very WIP bridge between Matrix and Discord in node.

# Features:

 - [x] Group messages
 - [ ] Direct messages
  - [ ] Recieving
  - [ ] Initiating
 - Matrix -> Discord
   - [x] Text content
   - [ ] Image content
   - [ ] Audio/Video content
   - [ ] Typing notifs
   - [x] User Profiles
 - Discord -> Matrix
   - [x] Text content
   - [ ] Image content
   - [ ] Audio/Video content
   - [x] Typing notifs
   - [ ] User Profiles
   - [ ] Presence
 - [ ] Webhooks (allows for prettier messages to discord)
 - [ ] Rooms react to Discord updates
 - [ ] Manage channel from Matrix
 - [ ] VOIP (**Hard** | Unlikely to be finished anytime soon)
